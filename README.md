# Bharat Fashion AI

Bharat Fashion AI is an AI-powered fashion recommendation web application built using Python and Streamlit. The platform allows users to explore different fashion categories and discover trending styles through an interactive interface.

## Features

* Fashion category browsing
* Interactive Streamlit interface
* Image-based category display
* E-commerce integration
* Responsive and user-friendly design

## Tech Stack

* Python
* Streamlit
* Git & GitHub

## Live Demo

(https://bharat-fashion-ai-lb366swykrvtuqjjt2eo6h.streamlit.app/)

## Installation

```bash
git clone <repository-url>
cd BHARAT-FASHION-AI
pip install -r requirements.txt
streamlit run app.py
```
