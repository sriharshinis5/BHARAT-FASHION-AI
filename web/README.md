# Bharat Fashion AI — Frontend

This is a static React frontend (no backend) implementing a Myntra-inspired UI and a small AI-powered recommendation and assistant using a local dataset.

Run locally (React dev):

```bash
cd web
npm install
npm start
```

Run static version directly in browser (no npm needed):

- Open `web/static/index.html` in your browser (double-click or drag into the browser).

Pages:
- Landing (home)
- Categories (recommendations)
- AI Assistant (chatbot)

Notes:
- Static version is at `web/static/` and uses the same embedded dataset.
- Recommendations link to Myntra product/category pages and open in a new tab.
