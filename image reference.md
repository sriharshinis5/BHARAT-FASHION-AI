## How to Download All Images

### Using Python (Download all images locally)

```python
import requests
from pathlib import Path

# Create assets folder
Path("assets").mkdir(exist_ok=True)

# Image mapping
images = {
    "kurti": "https://cdn.pixabay.com/photo/2023/02/14/11/05/ethnic-7787768_640.jpg",
    "dress": "https://cdn.pixabay.com/photo/2016/12/19/11/10/woman-1917497_640.jpg",
    "jeans": "https://cdn.pixabay.com/photo/2015/03/12/12/54/blue-jeans-668033_640.jpg",
    "shirt": "https://cdn.pixabay.com/photo/2016/12/26/17/28/man-1932847_640.jpg",
    "jacket": "https://cdn.pixabay.com/photo/2016/10/05/22/30/jacket-1716110_640.jpg",
    "footwear": "https://cdn.pixabay.com/photo/2018/02/27/05/40/shoes-3183358_640.jpg",
}

# Download
for name, url in images.items():
    response = requests.get(url)
    with open(f"assets/{name}.jpg", "wb") as f:
        f.write(response.content)
    print(f"✓ Downloaded {name}.jpg")
```
